## Lastpass2pass (NodeJS)

This is a simple NodeJS script I wrote to import my Lastpass data to pass.

### Usage:

```
# Clone repo
git clone https://github.com/YinkaEnoch/lastpass2pass.git

# Link repo
npm link

# Run script
lastpass2pass path/to/passwords_file.csv
```
